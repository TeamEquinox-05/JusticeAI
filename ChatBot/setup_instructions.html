AI Chatbot - Setup Instructions

Important: The chatbot application cannot run by simply opening the HTML file directly in a browser. It requires the Flask server to be running to handle API requests.

Why You're Seeing CORS or Fetch Errors

If you're seeing errors like Access to fetch at 'file:///C:/api/chat' has been blocked by CORS policy, it means you're trying to open the HTML file directly in a browser without running the Flask server.

How to Properly Run the Application

  1. Install the required Python packages: 
    pip install -r requirements.txt
  2. Make sure Ollama is running with the required embedding model: 
    ollama run mxbai-embed-large
  3. Create a .env file in the project root directory with your Google Gemini API key: 
    GEMINI_API_KEY=your_api_key_here
  4. Start the Flask server: 
    python app.py

    You should see output indicating the server is running on http://localhost:5000

  5. Open your web browser and navigate to: 
    http://localhost:5000

    This will load the chatbot interface properly, with all API endpoints connected to the backend server.

Troubleshooting

  • Make sure Flask is installed: pip install flask
  • If you see ModuleNotFoundError, install the missing package: pip install [package-name]
  • Ensure Ollama is running and the embedding model is available
  • Check that your .env file contains a valid Gemini API key

Once the server is running, access the chatbot at http://localhost:5000
